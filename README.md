# word_processor
A quick and fun excercise using vanilla JS to process markdown. 

Some markdown examples include:
HEADERS:
# This is an H1 #

## This is an H2 ##

### This is an H3 ######

HYPERLINKS
[This link](http://example.net/) has no title attribute.

ITALIC & BOLD
*single asterisks*

**double asterisks**
